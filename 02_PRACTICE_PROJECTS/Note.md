# Practice Projects Overview

This repository contains a collection of small web development practice projects.

## Project List

### [[PROJECT_01-NETFLIX/README|Netflix Clone]]
- **Description**: A static clone of the Netflix landing page focusing on layout and responsiveness.
- **Teaches**: CSS Flexbox/Grid, positioning, and building visually appealing UI layouts.

### [[PROJECT_02-CALCULATOR/index|Calculator]]
- **Description**: A fully functional arithmetic calculator.
- **Teaches**: DOM manipulation, event handling, and basic mathematical logic in JavaScript.

### [[PROJECT_03-FAULTY_CALC/index|Faulty Calculator]]
- **Description**: A calculator that occasionally gives wrong results to trick users.
- **Teaches**: Randomization logic, conditional statements, and debugging intentional bugs.

### [[PROJECT_04-PASS_GEN/index|Password Generator]]
- **Description**: A tool to generate secure, customizable passwords.
- **Teaches**: String manipulation, working with checkboxes, and the `Math.random()` function.

### [[PROJECT_05-COLOR_PALLETE/index|Color Palette Generator]]
- **Description**: Generates random color schemes for UI design.
- **Teaches**: Hexadecimal color codes, dynamic styling with JS, and clipboard APIs.

### [[PROJECT_05-HACKER_TERMINAL/index|Hacker Terminal]]
- **Description**: A simulated terminal interface with "hacking" animations.
- **Teaches**: Asynchronous programming (`async/await`), `setTimeout`, and terminal-like CSS styling.

### [[PROJECT_06-STOPWATCH/index|Stopwatch]]
- **Description**: A timer with start, stop, and reset functionality.
- **Teaches**: Time management with `setInterval`, formatting time strings, and state management.

### [[PROJECT_07-TODO_LIST/index|Todo List]]
- **Description**: A classic task management app with add/delete features.
- **Teaches**: Array methods, local storage (persistence), and CRUD operations in the UI.

### [[PROJECT_08-EXPENSE_TRACKER/index|Expense Tracker]]
- **Description**: Tracks income and expenses with a running balance.
- **Teaches**: Data filtering, calculating totals from arrays, and dynamic list rendering.

### [[PROJECT_09-TIC-TAC-TOE/index|Tic-Tac-Toe]]
- **Description**: A 2-player game on a 3x3 grid.
- **Teaches**: Winning algorithms, game loops, and managing a game board state.

### [[PROJECT_10-FLASHCARD_QUIZ/index|Flashcard Quiz]]
- **Description**: An interactive quiz tool for learning new concepts.
- **Teaches**: Object-based data structures, toggling visibility, and tracking user scores.

### [[PROJECT_11-TYPING_SPEED/index|Typing Speed Test]]
- **Description**: Measures words-per-minute (WPM) and accuracy.
- **Teaches**: String comparison, timing events, and calculating speed based on input length.

### [[PROJECT_12-WEATHER_DASHBOARD/index|Weather Dashboard]]
- **Description**: Displays real-time weather data using a public API.
- **Teaches**: Fetching API data, JSON parsing, and handling asynchronous requests.

### [[PROJECT_13-GITHUB_PROFILE_FINDER/index|GitHub Profile Finder]]
- **Description**: Searches and displays user info from the GitHub API.
- **Teaches**: API integration, template literals for UI rendering, and error handling for 404s.

### [[PROJECT_14-IMAGE_SEARCH/index|Image Search App]]
- **Description**: A gallery-style image search using Unsplash/Pixabay APIs.
- **Teaches**: Infinite scrolling or pagination, working with image data, and search query handling.

### [[PROJECT_15-DRAG&DROP_BOX/index|Drag & Drop Box]]
- **Description**: Interactive boxes that can be moved and rearranged.
- **Teaches**: The HTML5 Drag and Drop API and managing DOM event listeners for moving elements.
