# Mini JavaScript Projects Collection

A curated collection of 14+ mini web development projects built with HTML, CSS, and JavaScript. Each project is self-contained, beginner-friendly, and demonstrates a unique interactive feature or UI pattern. Perfect for learning, practice, and portfolio building.

---

## 📁 Projects Overview


This repository contains the following projects:

---

### 1. Calculator / Faulty Calculator
**Standard Calculator:** A basic calculator app supporting addition, subtraction, multiplication, and division with a clean UI.

**Faulty Calculator:** A fun twist—occasionally returns intentionally incorrect results by swapping operators randomly, simulating a "faulty" calculator.

---

### 2. Password Generator
Generate secure, random passwords with customizable options for length, uppercase, lowercase, numbers, and special characters. Ensures strong password creation for any use case.

---

### 3. Color Palette Generator
Create beautiful color palettes at the click of a button. Instantly generates random hex colors and updates the UI, making it easy to copy and use color codes for your designs.

---

### 4. To-Do List
A simple, interactive to-do/task manager. Add, edit, complete, and delete tasks. Tasks are saved in localStorage for persistence across sessions. Includes visual feedback for completed tasks.

---

### 5. Expense Tracker
Track your income and expenses with a clean UI. Add transactions, categorize as income or expense, and see your balance update in real time. Data is saved in localStorage for persistent tracking.

---

### 6. Flashcard Quiz App
Test your knowledge with interactive flashcards. Answer questions, get instant feedback, and track your score. Includes a variety of general knowledge questions and a progress tracker.

---

### 7. Typing Speed Test
Measure and improve your typing speed. Type randomly generated paragraphs, track your WPM (words per minute), CPM (characters per minute), mistakes, and time left. Great for practice and self-improvement.

---

### 8. Weather Dashboard
Get real-time weather information for any city using the OpenWeatherMap API. Displays temperature, weather conditions, and humidity. Includes error handling for invalid city names.

---

### 9. Tic-Tac-Toe
Classic tic-tac-toe game for two players. Features a responsive UI, win detection, and restart functionality. Great for quick fun and learning basic game logic.

---

### 10. Hacker Terminal
Animated, hacker-style terminal simulation. Displays random messages and animated dots to mimic a hacking sequence. Fun for creating a "hacker" effect in the browser.

---

### 11. Stopwatch
Simple stopwatch with start, stop, and reset functionality. Tracks hours, minutes, seconds, and milliseconds. Includes lap functionality for advanced timing needs.

---

### 12. Image Search
Search and display images from Unsplash using their API. Enter a search term, view image results, and load more images with pagination. Clean, responsive gallery layout.

---

### 13. GitHub Profile Finder
Lookup and display GitHub user profiles by username. Shows avatar, name, bio, followers, following, and public repositories. Handles errors for invalid usernames.

---

### 14. Drag & Drop Kanban Board
Organize tasks visually with a drag-and-drop Kanban board. Move tasks between columns, reorder within columns, and experience smooth drag-and-drop interactions for productivity.

---

## 🚀 Getting Started

You can run any project independently:

1. **Open directly**: Open the `index.html` file in your browser.
2. **Use Live Server**: Right-click `index.html` → "Open with Live Server" (VS Code extension)
3. **Python server**: Run `python -m http.server 8000` in the project folder
4. **Node.js server**: Run `npx http-server` in the project folder
